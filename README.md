<b>Installation</b>

To run this console app you need to have the following installed on your computer:
<b>.NET SDK.</b> for .NET CLI(command line interface). It is availible on https://dotnet.microsoft.com/en-us/download.This project was made using <b>.NET 8.0</b>.

Steps to install the project:

```
git clone https://github.com/DorianLeci/Internship-2-C-Sharp.git
cd Internship-2-C-Sharp-App
dotnet run
```
This app has 3 menus inside <b style="color:yellow">Main Menu</b>: <b>User Menu</b>,<b> Trip Menu </b> and StatisticsMenu </b>.

You can navigate through menus using numbers shown on each menu.Each number on those three menus leads to new submenu and so on.
Number 0 is always option for leaving child menu and go back to parent menu.If you are positioned in Main Menu by pressing 0 you can exit the app.
You can not skip menus(go from User Menu to Trip Menu directly).You can only go from parent to child directory and vice versa on step at a time.
